Local Tool Guide

tools/build_review_pack.py is a simple standard-library Python CLI that reads synthetic CSV inputs and generates a review pack plus findings logs.

Run

python tools/build_review_pack.py --input examples/sample-data.csv --output examples/sample-output.html --findings-csv examples/findings-log.csv --findings-jsonl examples/findings-log.jsonl

Boundaries

The tool classifies signals and generates sample outputs. It does not connect to live systems, send messages, approve work, change funding, change sequencing, assign owners, or accept risk.
