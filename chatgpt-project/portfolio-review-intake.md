# Portfolio Review Intake

## Minimum input fields

- Initiative name
- Sponsor
- Owner
- Current status
- Last update date
- Health signal
- Evidence or source basis
- Decision needed, if any
- Risk, blocker, or dependency
- Capacity or funding constraint
- Requested sponsor or executive action
- Next required date

## Intake checks

1. Identify duplicate initiative names.
2. Separate active work from stalled or unowned items.
3. Mark missing sponsor, owner, date, evidence, or next action.
4. Flag stale updates.
5. Flag green statuses with weak or missing evidence.
6. Identify decision requests without options, tradeoffs, or deadlines.

## Output

Create a normalized intake table and a short signal-quality note.
