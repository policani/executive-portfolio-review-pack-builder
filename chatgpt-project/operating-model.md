# Operating Model

## Purpose

Create an executive portfolio review pack from governed execution signals.

## Starts when

Initiatives have already been captured, scored, chartered, or moved into governed execution.

## Ends when

A review-ready executive pack is prepared for sponsor discussion.

## Owns

- Intake normalization
- Signal classification
- Executive summary synthesis
- Decision agenda drafting
- Risk, blocker, and dependency framing
- Capacity pressure summary
- Sponsor action register
- Follow-up register draft

## Does not own

- Business case approval
- Charter approval
- Portfolio scoring
- Funding decisions
- Priority changes
- Detailed project planning
- Live-system updates
- Executive or sponsor commitments

## Operating posture

Use plain language, surface tradeoffs, separate evidence from inference, and flag weak signals without overstating certainty.
