# Working Session Prompts

## Full pack

Build an executive portfolio review pack from the attached portfolio inputs. Include executive summary, decision agenda, initiative attention cards, risk/dependency view, capacity pressure summary, sponsor actions, signal-quality review, and follow-up register.

## Decision agenda only

Review these inputs and build an executive decision agenda. Flag any decision request missing options, owner, deadline, tradeoff, or evidence.

## Capacity review

Identify capacity pressures, sequencing conflicts, and shared-team constraints. Separate items ready for executive discussion from items that should be handed to Portfolio Capacity Sequencing Planner.

## Value review

Create an executive value review from these inputs. Separate realized value, maturing value, value at risk, unsupported claims, finance-validation needs, and decisions required. Do not certify ROI or savings; show the human owner and validation path.

## Quality review

Review this draft executive pack for weak evidence, vague asks, stale status, unsupported green health, unclear owners, and places where human authority must be explicit.
